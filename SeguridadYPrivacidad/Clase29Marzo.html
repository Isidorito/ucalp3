Servicios de Seguridad Informatica "bis"

  • -Anonimato -> Resguardo de datos y cuestiones legales que apliquen sobre eso
  • -Protección a la replica -> Princpio de Integridad -> Lo que llega es lo que se envió
  • -Confirmación de transacciónes -> Transacción: Todo que se debe ejecutar en su totalidad o cancelarse en su totalidad
  • -Referencia Temporal -> Validación de fechas para asegurar validez de datos
  • -Certificación mediante terceros -> Firmas/Certificados digitales

La seguridad informatica como un proceso

    >> PLAN -> DO -> CHECK -> ACT -> PLAN >>


  • Plan -> elegir las meidas y procedimientos a utilizar
  • Do -> Aplicación de las medidas
  • Check -> Comprobación de resultados y funcionamiento
  • Act -> Retroalimentación
