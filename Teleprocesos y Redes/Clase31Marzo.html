Definición: Conjunto de dispositivos comunicados entre sí en un medio particular

    Componentes
    • Dispositivos
    • Equipos de Comunicación
    • Medios (de transporte)
    • Adaptadores
    • Protocolos de envío y seguridad
    • Control de errores
    • Administración de recursos
    • Software adicional
    Caracteristicas
    • Información confiable / Replicación correcta de datos
    • Equipos capaces de identificarse entre sí
      • MAC
      • IP
      • URL
    • Manera standarizada de identificar las partes de la red
    Clasificación de Redes según la tecnología de transmisión utilizada
    • Redes de difución (Broadcasting)
      • Problema: Solapación o cuello de botella en el equipo transmisor
    • Redes Punto a Punto
      • Un enlace por cada par de elementos
      • Un nuevo elemento agregado requiere que se agregue la conexión a todos los otros elementos
    Clasificación según su Escala
    • PAN -> Personal Area Network
    • LAN -> Local Area Network
        • Topologías
        • Bus
        • Anillo
        • Estrella
        • Arbol
    • MAN -> Metropolitan Area Network
    • WAN -> WoldWide Area Network (Internet)
    • Metricas
    Protocolos de Comunicación -> Conjunto de Reglas para el intercambio de información
    • OSI -> Teorico
    • Definiciones
      • PDU -> Protocol Data Unit (todas las capas tienen)
      • Las Capas Transporte hasta Aplicación -> Puramente software
    • Capas
      • Capa Fisica
      • Capa de Enlace
      • Capa de Red -> Resolución de cuellos de botella
      • Capa de Transporte
      • Capa de Sesión
      • Capa de Presentación -> Convierte el tipo de dato a un generico para que puedan ser leidos en cualquier receptor
      • Capa de Aplicación -> Protocolo http(por ejemplo)
    • TCP/IP -> Implementación de OSI
      • Capa Enlace + Capa Fisica
      • Capa Red -> IP
      • Capa Transporte -> TPC y UDP
      • Capa Aplicación -> Capa Sesión + Capa Presentación + Capa de Aplicación
